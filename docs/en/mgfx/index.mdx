# MGFX

MGFX is the Lux rendering package for Garry's Mod immediate UI effects. It is
distributed as the standalone `lux-mgfx` package set and is documented in its
own site:

- [MGFX documentation](https://timewatcher.github.io/mgfx-docs-site/)
- [MGFX package repository](https://github.com/TimeWatcher/lux-mgfx)
- [MGFX documentation repository](https://github.com/TimeWatcher/mgfx-docs-site)

This page is only the Lux-site entry point. Use the standalone MGFX
documentation for API details, shader notes, performance guidance, and the
Lux-based architecture.

Install it per project:

```powershell
luxc install @lux/mgfx --from github:TimeWatcher/lux-mgfx
```

## Use from Lux

<LuxLuaPair title="Install the optional MGFX facade" luxLabel="Lux" luaLabel="Generated Lua">

```lux
import * as mgfx from "@lux/mgfx"
import * as console from "@lux/mgfx/console"

client fn installMgfx() {
  local api = mgfx.installGlobal("MGFX")
  console.install(api)
}
```

```lua
local mgfx = __lux_import("@lux/mgfx")
local console = __lux_import("@lux/mgfx/console")

local function installMgfx()
  local api = mgfx.installGlobal("MGFX")
  console.install(api)
end
```

</LuxLuaPair>

New Lux code should prefer explicit module imports such as `mgfx.paint`,
`mgfx.style`, `mgfx.frame`, and `mgfx.widgets`. The `MGFX.*` global facade is
for GLua-facing code, old panels, console demos, and gradual migration.

## What Lux Handles

`@lux/mgfx` is client-only. When a project imports it, `luxc gmod build`
compiles the required module parts, keeps realm exports client-scoped, emits
generated Lua, and generates the GMod loader. The package root comes from the
project lockfile after installation; it is not bundled into the default Lux
package submodule. Do not copy the old MGFX Lua autorun loader into a Lux
project.

MGFX is also a practical example of Lux package design: one package is split
into submodules such as `frame`, `paint`, `style`, `text`, `widgets`,
`materials`, and `shaderpack`, while module parts share internal scope and
export only the public API.
